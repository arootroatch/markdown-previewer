## Contributing

### Clone the repo

```bash
git clone https://github.com/arootroatch/js-calculator.git
cd js-calculator
```

### Install dependencies
```bash
npm install
```
### Submit a pull request
If you'd like to contribute, please fork the repository and open a pull request to the 'main' branch.
